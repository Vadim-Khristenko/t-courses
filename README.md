# T-Courses

LMS (Learning Management System) platform powering [algocourses.ru](https://algocourses.ru) and [t-edu.tech](https://t-edu.tech).

## Features

- Course management and content delivery
- Leaderboards and competitive programming standings
- Progress tracking for students
- Integration with external services (S3, MongoDB)
- Prometheus metrics for monitoring

## Tech Stack

- **Backend**: FastAPI, Uvicorn
- **Database**: MySQL (aiomysql), MongoDB (pymongo)
- **Templates**: Jinja2
- **Monitoring**: Prometheus

## Installation

```bash
pip install -e .
```
